# Diagnóstico de Marca - Compact

> Página de diagnóstico estratégico desenvolvida com a metodologia Intento para a Compact - Especialistas em máquinas em Cuiabá/MT.

## 🎯 Sobre o Projeto

Esta página representa uma análise crítica e estratégica da marca Compact, aplicando a metodologia proprietária da Intento para identificar contradições, oportunidades desperdiçadas e potencial de crescimento no mercado de máquinas agrícolas e industriais em Mato Grosso.

### Dados do Cliente
- **Empresa:** Compact
- **Contato:** Jader Amorim
- **Localização:** Cuiabá - MT
- **Especialização:** Máquinas (conforme Instagram @compactmaquinas)
- **Tempo de mercado:** 1-2 anos
- **Funcionários:** 2-5
- **Faturamento:** R$ 10-30k/mês
- **Orçamento disponível:** Até R$ 3k

### Análise Crítica Identificada
1. **Contradição Central:** Empresa especializada em máquinas compete como oficina genérica
2. **Mercado Subutilizado:** Localização privilegiada no hub do agronegócio brasileiro não é explorada
3. **Especialização Invisível:** @compactmaquinas sugere nicho, mas comunicação não reflete isso
4. **Capacidade Subestimada:** Estrutura robusta em pouco tempo indica competência não comunicada

## 🔧 Configuração & Deploy

### GitHub Pages (Principal)
1. Fazer push deste repositório para GitHub
2. Ativar GitHub Pages nas configurações
3. Selecionar branch main como source
4. A página estará disponível em: `https://[username].github.io/diagnostico-compact`

### Netlify (Backup)
1. Conectar repositório no Netlify
2. O arquivo `netlify.toml` já está configurado
3. Deploy automático será feito
4. URL personalizada pode ser configurada

### Assets Necessários
Para finalizar a implementação, solicitar:

1. **Logo da Compact** (formato SVG preferível)
   - Salvar como: `assets/compact-logo.svg`
   
2. **Foto da fachada** (alta resolução)
   - Salvar como: `assets/compact-fachada.jpg`
   
3. **Print do Instagram** (@compactmaquinas)
   - Para análise de conteúdo e estratégia

### Configuração de Analytics
1. Substituir `GA_MEASUREMENT_ID` no HTML e JS pelo ID real do Google Analytics
2. Configurar Facebook Pixel se necessário
3. Ajustar conversions tracking conforme objetivos específicos
